拖延症患者的第二序改变：责任、意志与行动

发表于 2016-11-24   |   分类于 mind hack

在行动科学中，系统内的改变是第一序改变，系统外的改变是第二序改变。如果我有拖延症，第一序改变是阅读所有拖延症相关的书籍，第二序改变是思考「拖延症更深层的问题」，以及如何解决它。

当我做第二序改变的思考时，我意识到自己不仅拖延任务，更拖延生活。

比如用五十天时间长了十五斤肌肉，过后却对甜食上瘾，攒下一肚子脂肪，脸都臃肿得不像话；比如用一个学期修完学分，却在暑假时沉迷游戏、把争取到的闲暇时间统统挥霍了；比如渴望婚姻和家庭，却无意中伤害了每一位和我在一起的姑娘。

我受够了现在的生活，是时候改变了。

对知识体系的探索（团队篇）

发表于 2016-11-14   |   分类于 mind hack

为什么需要知识体系？

因为好的知识体系可以将知识转化为生产力。

若没有知识体系，即便读完了芒格的书单，也无法成为芒格（志平仁波切语录）。芒格靠知识体系来读书，可以很快分辨出好书、烂书、低阶模型、高阶模型。和奥运选手一样，智者先分辨自己需要哪些营养，再摄取食物。

团队的成长和个人成长一样，需要优秀的模因和知识管理体系。如果把培养团队比作登山探险，那么优秀的模因是无坚不摧的开山斧，知识体系则是一份登山图。

六年前我曾带过一支团队，时至今日，仍对他们心怀愧疚，因为我只传递了模因，却没建立知识体系。其后果是，之后的每一任Leader，都要拼死努力，才能勉强维系团队水准。缺少登山图的他们，即便闯出了一百条山间小道，也难以到达山顶。所以这篇文章，想寄给六年前笨拙的自己。

如何阅读一本难啃的书&进化树：技术信息分析及新方案的产生（第1~2章）

发表于 2016-11-13   |   分类于 mind hack

这篇文章中我分享了大量卡片，事实上分享卡片是不可取的行为，因为它会让人不断思考，什么卡片应该公布，什么卡片应该只有自己看，浪费了大量时间。卡片和文章，应该是两个层级。

背景介绍

Triz理论曾被苏联列为高度机密、严禁外传，因为它揭示了创新的方法论，若流出国外，就等于给拿着柳叶刀攻城略地的敌人，送上一门红衣大炮。

所以读《创新算法》时，我才意识到读难书的意义：它能让我摆脱自己“原始人”的状态。我一直是发自内心地觉得自己什么都不懂，读了这本神作后，才稍微成长了一点点。

难书的精华有两种：反常识的证据和高阶模型，学到这两样东西有什么意义呢？

反常识的证据通常是科学家花费了大量金钱和时间研究出的成果。比如人类总是低估自己十年后会发生的变化（性格、喜好、职业生涯），因此常常活在过去的阴影中。如果不知道这个证据，你会常常责怪自己：我怎么变了？约翰•列侬的狂热的歌迷，也是因为不知道这个证据，一枪毙了一代奇才，还一脸嫌弃地说：你变了。

高阶模型的意义，就像阳老师所说的：在不同的维度上，分析对象的模型会发生不对称的变形，比如分子学和天体物理，这两门学科使用的是不同的模型，而高阶模型的意义在于，可以击穿这种变形，通用于各个维度上。比如牛顿的运动定律，可以适用于很多个维度，这件事情太美了。芒格的智慧，也是来自他的知识体系和两百多个高阶模型。

改变

每次读到一本好书，最强烈的感觉都是相见恨晚。为什么我没在高中时代，就阅读到这么撼人心魄的演讲集呢？
